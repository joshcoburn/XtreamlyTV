# XtreamlyTV 0.4.1 — Favorite collections

This release turns Favorites into a personalized, remote-first content hub while preserving the fast category-scoped provider browsers introduced in earlier releases.

## Highlights

- Favorites landing page with **All**, **Live TV**, **Movies**, and **Series** filter chips
- Built-in All Favorites, Live TV, Movies, and Series groups
- User-created collections with configurable names, icons, colors, and selected content
- Left-side group navigation with four-column channel and five-column poster grids
- Virtualized mixed-content Favorites grid
- Recently Watched Favorites based on actual viewing history
- Automatic custom-group cleanup when a favorite is removed
- Larger living-room typography and more sidebar icon-to-label spacing
- Red player key hint and improved menu-tooltip spacing

## Compatibility

Existing favorites migrate automatically. Custom collection metadata is stored locally on the television alongside existing settings, watch history, and resume positions.

## Testing

The webOS browser suite validates group creation data, group filtering, editor navigation, stale-membership cleanup, remote RED-button behavior, player tooltip styling, and 55,000-item catalog virtualization.

## Legal

XtreamlyTV includes no providers, subscriptions, playlists, channels, or copyrighted media.
