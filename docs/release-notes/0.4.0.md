# XtreamlyTV 0.4.0 — Multi-platform foundation

This is the first GitHub-ready monorepo release.

## Included

- Tested LG webOS app package
- Native Android TV / Google TV developer preview
- Shared contracts, brand assets, and design tokens
- Optional LAN API bridge
- CI builds for webOS and Android TV
- Tag-based GitHub release workflow
- Complete LG, TCL, Sony, emulator, ADB, build, install, and debugging documentation

## Platform maturity

- **webOS:** beta and physically tested through the v0.3.x iteration cycle
- **Android TV:** developer preview; compile and physical-device validation required before store distribution

No providers, channels, playlists, or copyrighted media are included.
