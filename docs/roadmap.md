# Roadmap

## 0.4 — Multi-platform foundation

- Monorepo and cross-platform contracts
- Android TV developer preview
- Media3 playback
- Shared design tokens and brand assets
- Platform build/install documentation
- CI and tag-based release automation

## 0.5 — Android feature parity

- Persistent favorites and recent history
- Editable provider settings
- Multiple skins
- Full Series season navigation
- Resume positions
- Player queue and channel switching polish
- Android TV diagnostics screen

## 0.6 — Guide and catch-up

- Full EPG grid
- Current/next program metadata
- Catch-up discovery and playback
- Live DVR-window controls where providers support them

## 0.7 — Profiles and safety

- Multiple providers
- Hidden/reordered categories
- PIN-protected categories
- User profiles
- Import/export settings without exporting plaintext passwords

## 1.0 — Stable television release

- webOS and Android TV feature parity
- Accessibility and localization review
- Automated upgrade/migration tests
- Signed Android release workflow
- Seller/store readiness documentation

## Future targets

- Samsung Tizen
- Fire TV packaging and store metadata
- Apple TV
- Desktop companion
- Mobile remote and keyboard companion
